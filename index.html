Command Function
git List Git commands

git checkout <branch> Switch branch to <branch>
git checkout -b <branch> Fork <branch> and switch on it
git branch -D <branch> Delete <branch> local
git push -d <remote> <branch> Delete <branch> on <remote> repository
git commit --allow-empty -m Commit without content, but with message
git commit Commit
git commit --amend Change last commit
git reset --hard && git clean -df Revert last commit
git diff <file> Compare changes of <file>
git pull Pull changes from remote repository
git pull <remote> <branch> Pull <branch> from a specific <remote> repository
git push Push changes to remote repository
git push <remote> <branch> Push changes to a specific <remote> repository <branch>
git stash Store changes in stash
git stash pop Get changes back out of stash
git stage . Load all changes to index
git add <file> Load changes of <file>s to index
git rm --cached Get changed files back to working tree
git add -p <file> Load parts of changes of <file> to index
y Stage hunk
n Do not stage hunk
q Quit, stage nothing
a Stage hunk with all following hunks
d Don't stage hunk and all following hunks too
g Select a hunk to go to
/ Search for a hunk
j Leave hunk undecided, see next undecided hunk
J Leave hunk undecided, see next hunk
k Leave hunk undecided, see previous undecided hunk
K Leave hunk undecided, see previous hunk
s Split hunk in smaller ones
e Edit current hunk
? Show help
git add -N <file> Add untracked <file> to repository
git diff --staged <file> Compare staged changes of <file>
git reset -p Get hunk back from index
git reset Get everything back from index
git restore Restore file in repository to its original state
